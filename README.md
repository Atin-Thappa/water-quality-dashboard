# Community Water Quality Monitoring Dashboard

A citizen-driven digital platform that allows users to easily report local water quality issues and visually communicate problem areas to municipal authorities.

## Problem Statement
In many parts of India, citizens face recurring issues with unsafe or poor-quality water supply, but lack an accessible, reliable, and low-effort way to report these issues to the appropriate municipal authorities.

## Tech Stack
### Frontend
- HTML, CSS, JavaScript
- Leaflet.js (for maps)
- Geolocation API

### Backend
- Python
- SQL (reports and locations tables)
